# Inox Nova Flutter App

This is a Flutter-based booking and QR attendance app with Razorpay integration.

## 🚀 Features
- 🔐 Login / Signup with Firebase
- 📅 Booking management
- 📷 QR Code generation and scanning
- 💳 Razorpay payment integration
